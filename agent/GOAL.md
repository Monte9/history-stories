# Goal

A growing library of short, well-written historical stories spanning Roman history, the Ramayana, and the Mahabharata. Each story teaches real history with specific dates, names, and context, paired with cinematic AI-generated cover art. Browsable by tradition, with character profiles, timelines, and daily quotes. The kind of site you'd send to a friend who's getting into history.

Dark theme, purple/violet accent, tradition-colored tags (amber/Roman, emerald/Ramayana, blue/Mahabharata). Vivid but not corny. Quality over feature count.

This file is human-owned. The planner reads it; agents never edit it.
