# History Stories — Plan

Historical stories with cinematic cover art, published to the web. Generated by Ash, directed by Monte.

---

## Vision

A growing library of short, well-written historical stories spanning Roman history, the Ramayana, and the Mahabharata. Each story teaches real history with specific dates, names, and context, paired with cinematic AI-generated cover art. Browsable by tradition, searchable, and shareable. The kind of site you'd send to a friend who's getting into history.

---

## Current State

**Live at [historystories.vercel.app](https://historystories.vercel.app).** Auto-deploys from `nexuslabsx/history-stories` on push.

- 7 stories published (3 Ramayana, 2 Roman, 2 Mahabharata)
- Characters covered: Scipio Aemilianus, Karna, Jatayu, Marcus Aurelius, Hanuman, Rama, Ashwatthama
- Each story has AI-generated cover art (Gemini Pro via nano-banana skill)
- File-system architecture: markdown stories + frontmatter, git = database, static export
- Next.js 16 + Tailwind 4, dark theme with purple/violet accent
- Story cards with tradition-colored tags (amber/Roman, emerald/Ramayana, blue/Mahabharata)
- Individual story pages with hero cover images
- Cross-harness pipeline: both OpenClaw and Cursor can generate and publish stories
- `history.json` tracker prevents character/type repetition across generations
- `story-from-history` skill handles the full pipeline: generate text + cover art, write files, commit, push, deliver

## Phases

### Phase 1: Content Depth
- Batch-generate 10-15 more stories to reach ~20+ total
- Ensure balanced tradition coverage (aim for roughly equal Roman/Ramayana/Mahabharata)
- Expand character range: Seneca, Augustus, Cincinnatus, Draupadi, Bhishma, Krishna, Vibhishana, Bharata
- Vary content types: more origin stories, breakthroughs, timelines (currently heavy on hard lessons and character studies)

### Phase 2: Browse & Filter UX
- Add tradition filter tabs or toggle on the homepage (All / Rome / Ramayana / Mahabharata)
- Add content type tags to story cards
- Sort options: newest first (current), oldest first, by tradition
- Basic search or keyword filter across titles and one-liners
